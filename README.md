# Zum Starten der App

## Voraussetzungen

- Alle Schüler im selben Netzwerk wie der Host (du)
- Docker installiert & geöffnet
- Für den QR Code:
  - Python3 installiert (Windows)

    ⚠️ Bei der Python-Installation unter Windows unbedingt "Add Python to PATH" anhaken.
  - Homebrew (macOS)

## Start

### 1. Windows

```start.bat``` öffnen

### 2. Mac / Linux

```bash
cd path/to/project
./start.sh
```
## Funktionsweise

Die App startet in einer Fake-Ilias Seite, in welcher sich die Schüler anmelden sollen.
Dann werden sie weitergeleitet auf unseren Kurs, welcher sie interaktiv über die Sicherheit im Internet lehrt.
