# Zum Starten der App

## Voraussetzungen

- Alle Schüler im selben Netzwerk wie der Host (du)
- Docker installiert & geöffnet
- Für den QR Code:
  - Python3 installiert (Windows)

    ⚠️ Bei der Python-Installation unter Windows unbedingt "Add Python to PATH" anhaken.
  - Homebrew (macOS)

## 1. Windows

```start.bat``` öffnen

## 2. Mac / Linux

```start.sh``` öffnen
